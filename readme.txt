Partyzone Launcher
==================

Управление программой
---------------------

Alt + C Показать или скрыть окно игрового чата

Сборка под управлением Windows
------------------------------

1.  Нажмите правой кнопкой мыши на файл проекта partyzone_launcher.pro.
    В появившемся меню выберите пункты "открыть консоль в этом каталоге"
    -> "среда сборки".

2.  Отредактируйте скрипт deploy-windows-mingw.bat, заменив указание
    каталога C:\Qt\5.15.2\msvc2015_64\ на свой каталог установки Qt

3.  Запустите скрипт deploy-windows-mingw.bat. Он создаст папку
    distrib, содержащую исполнимый файл и все необходимые для его
    работы библиотеки
